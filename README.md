# Toward Safer Video Platforms: Leveraging Large Vision-Language Models for Multimodal Sentiment Analysis

This repository contains the data, evaluation rubric, and analysis materials supporting the paper:

**Toward Safer Video Platforms: Leveraging Large Vision-Language Models for Multimodal Sentiment Analysis**  
This paper is currently under review.

---

## What is inside this repo?

- Jupyter notebooks for Multimodal Sentiment Analysis baseline analysis  
- Supporting materials to enable reproducibility and further extension of the analysis
- Text transcriptions both models  

---

## Models Evaluated

- LLaVA-One-Vision-7B 
- Qwen2.5-VL-7B
